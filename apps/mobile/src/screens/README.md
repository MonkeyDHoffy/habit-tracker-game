# Screen Plan

- `home`: Startbildschirm und Einstieg.
- `run`: Aktiver Run, Tagesstatus und Hauptaktionen.
- `battle`: Kampfansicht gegen den Schweinehund.
- `routines`: Routinen verwalten und abhaken.
- `stats`: Fortschritt, Trends, Streaks.
- `profile`: Spielerprofil, Avatar, Präferenzen.
- `settings`: App-Einstellungen.
- `help`: FAQs und kurze Erklärungen.
- `onboarding`: Erste Einrichtung beim App-Start.
